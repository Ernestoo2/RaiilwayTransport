## Setup and Installation

### Prerequisites
- Node.js (v16+)
- npm or Yarn

### Installation Steps
1. Clone the repository
```bash
git clone https://github.com/your-username/nft-marketplace.git
```

2. Install Dependencies
```bash
npm install
# or
yarn install
```

3. Run Development Server
```bash
npm start
# or
yarn start
```
